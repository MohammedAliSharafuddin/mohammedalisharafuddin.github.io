02 · Web Tools

Standalone browser-based tools built for quick, useful decisions. Price Memory turns your own purchase log into a grocery price baseline, Margin Guard checks whether a promotion can recover the contribution margin it gives away, and Persona Builder turns research notes into a shareable persona card.

Browse web tool repositories on GitHub →

Persona Builder

Live

A single-page tool for building UX research and marketing personas at three levels of detail, basic, standard, and full, with photo or sample avatar selection and one-click export to PNG or PDF.

UX ResearchMarketingLocal StorageVanilla JS
Web Application Open App →

Price Memory

Live

A local grocery price tracker built as a single-page web app. It stores item, store, price, and date in IndexedDB so you can build a personal price history and check whether a sale is actually below your usual price.

PWAIndexedDBCanvas APIVanilla JS
Web Application Open App →

Margin Guard

Live

A single-page promotion break-even calculator for pricing and campaign decisions. It compares baseline contribution against promo-period contribution and shows discount depth, promo contribution per unit, break-even units, required unit lift, and expected net contribution delta.
