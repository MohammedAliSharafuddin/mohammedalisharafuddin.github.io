About

Senior Lecturer at the Qasim Ibrahim School of Business, Villa College, Malé, Maldives. This site brings together research outputs, textbooks, templates, and browser-based tools for management, marketing, and statistics. Current work centres on practical analytics, open teaching materials, and decision-support tools designed for direct classroom use and independent study.

Web Tools

Small browser-based utilities built for practical teaching and everyday decision support. The current collection includes Price Memory for local grocery price tracking and Margin Guard for promotion break-even analysis.

Browse tool source on GitHub →

Price Memory

Live

A local grocery price tracker built as a single-page web app. It stores item, store, price, and date in IndexedDB so you can build a personal price history and check whether a sale is actually below your usual price.

PWAIndexedDBCanvas APIVanilla JS
GitHub Pages App Open App →

Margin Guard

Live

A single-page promotion break-even calculator for pricing and campaign decisions. It compares baseline contribution against promo-period contribution and shows discount depth, promo contribution per unit, break-even units, required unit lift, and expected net contribution delta.
